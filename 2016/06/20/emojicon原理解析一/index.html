EmojiconFragment。emojicon项目地址https://github.com/rockerhieu/emojicon

二. emojicon library代码结构

library 项目结构如图所示/librarystru.png)
emoji文件夹下的类：

  1. Emojicon为整个项目对应表情的数据结构，继承了Parcel接口，方便在整个项目中传递。Emojicon的代码结构如下：/Emojiconstruc.png)

  2. Nature/Objects/people/places/symbols对应表情的几个分类，分别建立对应分类的Emojicon 数组,并且通过Emojicon.fromCodePoint或者fromchar来初始化Emojicon

    public static Emojicon fromCodePoint(int codePoint) {

    Emojicon emoji = new Emojicon();
emoji.emoji = newString(codePoint);
return emoji;

    }

    public static Emojicon fromChar(char ch) {

    Emojicon emoji = new Emojicon();
emoji.emoji = Character.toString(ch);
return emoji;

    }
    通过这二个函数可以将表情对应的Unicode编码十六进制，转换成其存储在 char 数组中的 UTF-16 表示形式。

  3. 所有的IM类的应用都会用到自定义的表情输入，也就是类似于下图所示效果/fragment.png)

下面来详细解读整个EmojiconsFragment,首先是实列化函数

     public static EmojiconsFragment newInstance(boolean useSystemDefault) {
EmojiconsFragment fragment = new EmojiconsFragment();
Bundle bundle = new Bundle();
bundle.putBoolean(USE_SYSTEM_DEFAULT_KEY, useSystemDefault);
fragment.setArguments(bundle);
return fragment;
}

useSystemDefault 这个变量用来选择是否使用系统自带表情。

onCreateView函数,初始化EmojisPagerAdapter，需要一个fragment的数组
mEmojisAdapter = new EmojisPagerAdapter(getFragmentManager(), Arrays.asList(EmojiconRecentsGridFragment.newInstance(mUseSystemDefault), EmojiconGridFragment.newInstance(People.DATA, recents, mUseSystemDefault), EmojiconGridFragment.newInstance(Nature.DATA, recents, mUseSystemDefault), EmojiconGridFragment.newInstance(Objects.DATA, recents, mUseSystemDefault), EmojiconGridFragment.newInstance(Places.DATA, recents, mUseSystemDefault), EmojiconGridFragment.newInstance(Symbols.DATA, recents, mUseSystemDefault) ));
提交了6个fragment，并且对每个view设置监听。
EmojiconGridFragment onCreateView加载emojicon_grid 创建一个View：
inflater.inflate(R.layout.emojicon_grid, container, false)
onViewCreated函数对gridview设置了一个Adapter:

gridView.setAdapter(new EmojiAdapter(view.getContext(), mData, mUseSystemDefault));

EmojiAdapter:
EmojiAdapter里面的Holder只有一个控件EmojiconTextView, 该控件通过获取Emojicon里面的emoji字段来显示。这个自定义的TextView后续再介绍。
